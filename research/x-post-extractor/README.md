# X投稿抽出 最小検証（Claude非使用）

note-writer本体・DBとは無関係の、独立した検証用スクリプトです。
このディレクトリの内容はどこからもimportされておらず、アプリの動作には一切影響しません。

## 目的

Claude in Chrome / Fableを実行時に使わず、通常のJavaScriptだけで
X検索結果ページから投稿情報を取得できるかを確認する。

## やらないこと（重要）

- Xへの自動ログイン
- 自動スクロール・自動クリック（「さらに表示」も含む）
- 定期実行・繰り返し実行
- ネットワーク通信（外部送信・API呼び出し）
- いいね・返信・リポスト・フォロー・投稿・DM・設定変更
- ログイン情報やCookieの読み出し・保存
- note-writor本体・DBへの保存

## 実行手順

1. リサーチ専用Xアカウントでログイン済みの状態で、検索結果の「最新」タブを開く
2. 検証対象の投稿5件が画面に表示されていることを確認する（スクロール不要な範囲）
3. ブラウザのDevToolsを開き、Consoleタブを選択する
4. `extract-x-posts.js` の内容全体をコピーし、Consoleに貼り付けてEnter
5. コンソールに抽出結果のJSONが表示される（`window.__xExtractResult` にも格納される）
6. 結果を確認したい方法に応じて、続けて次のいずれかをConsoleで実行する
   - クリップボードにコピーしたい場合：`copyXExtractResultToClipboard()`
   - JSONファイルとしてダウンロードしたい場合：`downloadXExtractResultAsJson()`

## 成功条件の確認方法

同じ検索画面を対象に、Claude in Chromeで取得した5件の結果と本スクリプトの
出力を突き合わせ、以下をすべて確認してください。

- [ ] 5件すべての `postId` と `url` が一致する
- [ ] `text`（本文）と `authorName`（投稿者表示名）・`authorHandle`（@ハンドル）が一致する
- [ ] `replies` / `reposts` / `likes` / `bookmarks` / `views` の5種類が一致する
- [ ] 他の投稿の数値が混ざっていない（articleごとに閉じたスコープで取得しているか）
- [ ] 実行中、Networkタブに新規リクエストが発生していない
- [ ] Claude in Chrome・Fable・その他AIを一切使わずに実行できた

## 反応数の解析方法

`role="group"` の `aria-label` 文字列から、出現順ではなく **項目名（キーワード）**
を手がかりに各指標の数値を抽出します。判定するキーワードは以下の通りです
（日本語UI優先、見つからない場合は英語表記も試す）。

| 指標 | 日本語キーワード | 英語キーワード |
|---|---|---|
| replies | 返信 | reply / replies |
| reposts | リポスト | repost / reposts |
| likes | いいね | like / likes |
| bookmarks | ブックマーク | bookmark / bookmarks |
| views | 表示 | view / views |

いずれのキーワードにも一致しなかった指標は `null` になります。項目が1つ欠けて
いても、他の項目の値がずれて割り当てられることはありません（出現順には一切
依存しないため）。

## 既知の制約・未検証事項

- `authorHandle` は実際にはXの内部的な数値ユーザーIDではなく、プロフィールURLから
  取得した **@ハンドル（screen name）** です。数値IDはDOM上のどこにも表示されない
  ため取得できません。
- 上記キーワードによる解析は、実際のX日本語UI・英語UIの`aria-label`実文言に対して
  一度も検証していません。文言が想定と異なる場合、該当指標は `null` のままに
  なります（誤った数値を作らないための安全側の挙動です）。
- 表示回数などが「12.3万」「12.3K」のような省略表記になっている場合、
  このスクリプトは正規化せず `null` を返します（誤った数値を作らないため）。
- 本スクリプトは実際のXページに対して一度も実行しておらず、静的なコードレビュー
  のみの状態です。実機での動作確認と、Claude in Chromeの結果との突き合わせが
  必須です。
